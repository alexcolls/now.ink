## Introduction

now.ink ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application and services.

**By using now.ink, you agree to the collection and use of information in accordance with this policy.**

## Information We Collect

### 1. Wallet Information
- **Solana Wallet Address:** Used for authentication and NFT ownership
- **Public Key:** Displayed on your profile and associated with your content
- **Transaction History:** Publicly visible on Solana blockchain

### 2. Location Data
- **GPS Coordinates:** Captured when recording videos
- **Embedded in NFTs:** Permanently stored with minted content
- **Map Display:** Used to show your moments on the public map

### 3. Video Content
- **Recorded Videos:** Stored on Arweave (permanent, decentralized)
- **Thumbnails:** Generated for display purposes
- **Metadata:** Title, timestamp, duration associated with videos

### 4. Usage Data
- **API Requests:** Server logs for debugging and analytics
- **Device Information:** OS version, app version for support
- **Crash Reports:** Anonymized error logs for bug fixes

### 5. Optional Information
- **Username:** If you choose to set one
- **Bio:** Profile description (optional)
- **Avatar:** Profile picture (optional)

## Data Storage and Security

### Blockchain Storage (Permanent)
- **NFT Metadata:** Stored on Solana blockchain (immutable)
- **Video Content:** Stored on Arweave (permanent, decentralized)
- **GPS Coordinates:** Embedded in NFT metadata (public)

⚠️ **Important:** Once an NFT is minted, the data cannot be deleted or modified.

### Our Servers (Temporary)
- **Video Files:** Temporarily cached during upload process
- **User Profiles:** Stored in PostgreSQL database
- **Session Data:** JWT tokens with 24-hour expiration

### Security Measures
- ✅ **Encryption in Transit:** All API calls use HTTPS
- ✅ **Database Encryption:** PostgreSQL with encrypted backups
- ✅ **Access Control:** Role-based permissions
- ✅ **Secure Wallets:** Private keys never leave your device
- ✅ **Rate Limiting:** Protection against abuse
